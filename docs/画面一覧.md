# 画面一覧

- 学習者ログイン画面学習者
  - メールアドレス・パスワードで認証
  - Student / 管理者のロール判定

- 学習者ダッシュボード学習者
  - 学習記録の総学習時間・平均学習時間表示
  - 学習カテゴリ別色分け（自動着色）
  - 自己PR生成ボタン
  - Studentは自分のデータのみ、管理者は所属会社の全Studentデータを閲覧

- 学習者学習カテゴリ管理画面学習者
  - 自由にカテゴリを作成・更新・削除
  - Studentは自分専用のカテゴリを管理

- 学習者学習記録画面学習者
  - 学習記録の作成・閲覧・更新・論理削除
  - 学習カテゴリ選択（自由作成したカテゴリ）
  - 学習時間入力（小数点で2.5hなど）
  - メモ入力

- 学習者日報画面学習者
  - 日報の作成・閲覧・更新・論理削除
  - 今日の目標、達成したこと、苦戦したこと、乗り越え方を入力

- 学習者自己PR生成画面学習者
  - 学習記録から100文字程度で自己PR生成
  - 編集は自分のPRのみ可能
